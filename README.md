This project analyzes MSME registration trends in Rajkot district using Power BI and government open data.

## Objective
To understand MSME growth, sector distribution, and area-wise concentration in Rajkot.

## Key Insights
- MSME registrations increased by **118%** from 2020 to 2025
- Manufacturing sector contributes **57%** of total MSMEs
- Kothariya, Gondal Road, and 150 FT Ring Road have the highest MSME concentration

## Tools Used
- Power BI
- Azure Maps
- Excel
- Data Source: data.gov.in
